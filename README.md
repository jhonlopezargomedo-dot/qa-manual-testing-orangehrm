# 🧪 QA Manual Testing - OrangeHRM

## 📌 Descripción
Proyecto de pruebas manuales enfocado en el módulo de login del sistema OrangeHRM.

## 🎯 Objetivo
Validar el correcto funcionamiento del proceso de autenticación mediante pruebas funcionales y negativas.

## 🧪 Alcance de pruebas
- Login con credenciales válidas
- Login con credenciales inválidas
- Validación de campos obligatorios
- Manejo de sesión
- Acceso sin autenticación

## 📋 Casos de prueba
Se diseñaron y ejecutaron más de 10 casos de prueba basados en criterios de aceptación.

## 🐞 Bugs encontrados
Se identificaron múltiples errores relacionados con:
- Validación de credenciales
- Mensajes incorrectos
- Validación de inputs

## 🛠️ Herramientas
- Excel / Google Sheets
- Navegador web
